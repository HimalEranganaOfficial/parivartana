# ПАРИВАРТАНА (පරිවර්තන)

> *Parivartana — "Translation" in Sinhala (සිංහල)*

An intelligent system that automates layout-preserving document translation for professional translators.

## The Problem

Professional translators produce **page-to-page translations** — translated text must be placed in the exact same position as the original document. This is currently done manually in MS Word using grid-based tables with merged cells, a process that is extremely time-consuming.

## The Solution

**Parivartana** automates the entire workflow:

1. **Scan** — Takes a scanned document image as input
2. **Detect** — Identifies text regions and their positions using OCR
3. **Translate** — Routes text to the best AI for the language pair
4. **Reconstruct** — Generates an output document with translated text in the same layout as the original

## Target Languages

- Sinhala (සිංහල)
- English
- Russian (Русский)
